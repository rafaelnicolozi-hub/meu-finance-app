# Finance Tracker

App simples de controle financeiro pessoal, feito em React como projeto de estudo.

## O que o app faz

- Cadastro de transações com descrição, valor e tipo (receita/despesa)
- Remoção de transações
- Filtro por Todas / Receitas / Despesas
- Cores diferentes para receita (verde) e despesa (vermelho)
- Dados salvos no navegador (localStorage) — não perde ao atualizar a página

## Tecnologias

- React
- Vite
- Tailwind CSS

## Como rodar

```bash
npm install
npm run dev -- --port 4000
```

Depois acesse `http://localhost:4000` no navegador.
